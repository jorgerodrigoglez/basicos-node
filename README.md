# Notas:

Básico de NODE

```
--help     Muestra ayuda                                        [booleano]
--version  Muestra número de versión                            [booleano]
-b, --tabla    es la tabla de multiplicar que quieres listar
                                                            [número] [requerido]
-h, --hasta    hasta que numero quieres multiplicar     [número] [defecto: 10]
-l, --listar   es la opcion para listar la tabla de multiplicar
                                                     [booleano] [defecto: false]
```
